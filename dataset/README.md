# Датасеты для обучения адаптеров

## Использование

| Файл | Адаптер | Статус |
|------|---------|--------|
| `russian_gec_dataset_final (1).csv` | Нормативность | Используется |
| `LORuGEC.xlsx` | Нормативность | Используется |
| `in_domain_train.csv` | Нормативность | Используется |
| `LiDiRus.jsonl` | Логика | Используется |

**Релевантность** — отдельный файл не нужен: адаптация при загрузке лекции к шаблону.

**Корректность** — датасет пока отсутствует (zero-shot Mistral).

## Обучение

```powershell
.\training\run_training.ps1
```

Результаты: `ai-service/models/adapters/*.joblib`, метрики в `training/reports/`.
